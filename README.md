# bitcron-theme-next

此主题在 bitcron 开源主题 [next](https://blog-next--theme.bitcron.com) 的基础上进行个性化修改，用于个人博客，[博客地址](https://ahacker.bitcron.com)。


## 改造

- [x] 图片样式优化：加圆角和阴影
- [x] 移动端访问文章详情页面边距减小
- [x] 首页每页展示 10 篇
- [ ] 仿 Medium 文章列表页样式
- [ ] 首页左边增加分类快速导航
- [ ] 文章预览时包含图片显示过大
- [ ] 美化分类、标签页面
- [x] 代码高亮及容器样式美化
